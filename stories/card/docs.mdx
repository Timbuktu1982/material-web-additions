## Example Usage
### Standard
<Canvas
    withToolbar
    mdxSource={dedent`
        <mwa-card>
          Content
        </mwa-card>
    `}>
    <mwa-card style={{'--mdc-card-padding': '16px'}}>
        Content
    </mwa-card>
</Canvas>

> You may want to apply padding to card contents (recommended to `16px` by the Material Design guidelines).
> In the previous live example and the following one the padding is already applied.
> ```css
> mwa-card {
>     --mdc-card-padding: 16px;
> }
> ```

### Outlined card
Just add the `outlined` attribute to your card.

<Canvas
    withToolbar
    mdxSource={dedent`
        <mwa-card outlined>
          Content
        </mwa-card>
    `}>
    <mwa-card outlined style={{'--mdc-card-padding': '16px'}}>
        Content
    </mwa-card>
</Canvas>

### Primary action
If a majority of the card should be actionable, you can wrap the actionable content inside the `<mwa-card-primary-action>` element.

<Canvas
    withToolbar
    mdxSource={dedent`
        <mwa-card class="demo-card">
            <mwa-card-primary-action>
                <div class="demo-card__header">
                    <div class="demo-card__header-text">
                        <div class="demo-card__title">Our Changing Planet</div>
                        <div class="demo-card__subtitle">by Kurt Wagner</div>
                    </div>
                </div>
            </mwa-card-primary-action>
        </mwa-card>
    `}>
    <mwa-card class="demo-card">
        <mwa-card-primary-action>
            <div className="demo-card__header">
                <div className="demo-card__header-text">
                    <div className="demo-card__title">Our Changing Planet</div>
                    <div className="demo-card__subtitle">by Kurt Wagner</div>
                </div>
            </div>
        </mwa-card-primary-action>
    </mwa-card>
</Canvas>

### Media
If you want to display an image in the card, you can add the `<mwa-card-media>` component to your card/primary action.

<Canvas
    withToolbar
    mdxSource={dedent`
        <mwa-card class="demo-card">
            <mwa-card-primary-action>
                <mwa-card-media aspectRatio="16-9" class="demo-card__media demo-crop-image"></mwa-card-media>
                <div class="demo-card__header">
                    <div class="demo-card__header-text">
                        <div class="demo-card__title">Our Changing Planet</div>
                        <div class="demo-card__subtitle">by Kurt Wagner</div>
                    </div>
                </div>
                <div class="demo-card__secondary mdc-typography mdc-typography--body2">
                    Visit ten places on our planet that are undergoing the biggest changes today.
                </div>
            </mwa-card-primary-action>
        </mwa-card>
    `}>
    <mwa-card class="demo-card">
        <mwa-card-primary-action>
            <mwa-card-media aspectRatio="16-9" class="demo-card__media demo-crop-image"></mwa-card-media>
            <div className="demo-card__header">
                <div className="demo-card__header-text">
                    <div className="demo-card__title">Our Changing Planet</div>
                    <div className="demo-card__subtitle">by Kurt Wagner</div>
                </div>
            </div>
            <div className="demo-card__secondary mdc-typography mdc-typography--body2">
                Visit ten places on our planet that are undergoing the biggest changes today.
            </div>
        </mwa-card-primary-action>
    </mwa-card>
</Canvas>


### Actions
This area is used to show different actions the user can take, typically at the bottom of a card.

#### Buttons
You can add buttons in your card via the `actionButtons` slot:
<Canvas withToolbar>
    <Story id="card--basic-with-only-buttons"/>
</Canvas>

#### Icon Buttons
You can add icon buttons in your card via the `actionIcons` slot:

## API
### Slots
| Name     | Description                                                        |
|----------|--------------------------------------------------------------------|
| default  | Default slot that contains the card contents and actions           |
| `button` | Optional `<button>` to display an action button in the action row. |
| `icon`   | Optional `<button>` to display an action button in the action row. |
