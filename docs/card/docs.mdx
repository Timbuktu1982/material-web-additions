### Outlined card
To use an outlined card, use the `md-outlined-card` element.
<Canvas of={CardStories.SimpleOutlined} withToolbar/>

### Filled card
To use a filled card, use the `md-filled-card` element.
<Canvas of={CardStories.SimpleFilled} withToolbar/>

### Primary action
If most of the card should be actionable, you can add the `clickable` attribute to your card element.

<Canvas of={CardStories.PrimaryActionDemo} withToolbar/>

### Media
If you want to display an image in the card, you can add an image with a standard `<img>` element.

<Canvas of={CardStories.MediaDemo} withToolbar/>

### Actions
This area is used to show different actions the user can take, typically at the bottom of a card.

#### Buttons
You can add buttons in your card via the `button` slot:

<Canvas of={CardStories.BasicWithOnlyButtons} withToolbar/>

#### Icon Buttons
You can add icon buttons in your card via the `icon` slot:

<Canvas of={CardStories.BasicWithOnlyIcons} withToolbar/>

Other examples are available in the sidebar or below these docs.

## API
### Slots
| Name     | Description                                                        |
|----------|--------------------------------------------------------------------|
| default  | Default slot that contains the card contents and actions           |
| `button` | Optional `<button>` to display an action button in the action row. |
| `icon`   | Optional `<button>` to display an action button in the action row. |
